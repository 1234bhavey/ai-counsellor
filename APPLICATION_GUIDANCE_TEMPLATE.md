## Application Requirements Framework

### Document Categories

#### 1. Academic Documents
- **Transcripts**: Official academic records from all institutions
- **Degree Certificates**: Completed degree documentation
- **Grade Conversion**: GPA/percentage conversion if required
- **Course Descriptions**: Detailed syllabus for credit evaluation

#### 2. Test Scores
- **English Proficiency**: IELTS/TOEFL scores
- **Standardized Tests**: GRE/GMAT scores (program dependent)
- **Subject Tests**: GRE Subject tests for specific programs
- **Test Score Reports**: Official score reports from testing agencies

#### 3. Application Essays
- **Statement of Purpose (SOP)**: Personal academic and career statement
- **Personal Statement**: Character and motivation essay
- **Essays**: Program-specific essay questions
- **Research Proposal**: For research-based programs

#### 4. Recommendation Letters
- **Academic References**: From professors/academic supervisors
- **Professional References**: From employers/supervisors
- **Character References**: Personal character attestations
- **Reference Forms**: University-specific recommendation forms

#### 5. Financial Documents
- **Bank Statements**: Proof of financial capacity
- **Scholarship Letters**: Award notifications if applicable
- **Sponsor Letters**: Financial sponsorship documentation
- **Tax Returns**: Income verification documents

#### 6. Identity & Legal Documents
- **Passport**: Valid passport with sufficient validity
- **Visa Documents**: Student visa application materials
- **Medical Records**: Health clearance certificates
- **Police Clearance**: Background check certificates

### Timeline Framework

#### Phase 1: Foundation (Weeks 1-4)
**Focus**: Document gathering and test preparation
- Collect academic transcripts and certificates
- Register for required standardized tests
- Begin SOP drafting process
- Identify and contact recommenders

#### Phase 2: Content Creation (Weeks 5-8)
**Focus**: Essay writing and application completion
- Complete SOP and personal statements
- Finalize recommendation letter requests
- Prepare financial documentation
- Complete application forms

#### Phase 3: Submission & Follow-up (Weeks 9-12)
**Focus**: Application submission and tracking
- Submit completed applications
- Follow up on recommendation letters
- Track application status
- Prepare for interviews if required

### Task Generation Logic

#### Priority Levels
- **Critical**: Must complete for application submission
- **Important**: Significantly impacts application strength
- **Recommended**: Enhances application but not mandatory

#### Time Estimates
- **Quick (1-2 hours)**: Form filling, online registrations
- **Medium (1-2 days)**: Document requests, basic research
- **Long (1-2 weeks)**: Essay writing, test preparation

#### Dependencies
- **Sequential**: Must complete A before starting B
- **Parallel**: Can work on simultaneously
- **Conditional**: Only required if certain conditions met

## Timeline Calculation Logic

### Deadline-Based Planning
```javascript
const calculateTimeline = (applicationDeadline, currentDate) => {
  const weeksAvailable = Math.floor((applicationDeadline - currentDate) / (7 * 24 * 60 * 60 * 1000));
  
  if (weeksAvailable >= 12) {
    return 'comfortable'; // Full 12-week timeline
  } else if (weeksAvailable >= 8) {
    return 'moderate'; // Compressed 8-week timeline
  } else if (weeksAvailable >= 4) {
    return 'urgent'; // Rush 4-week timeline
  } else {
    return 'critical'; // Emergency timeline
  }
};
```
